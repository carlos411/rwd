# 6.3 transition-duration

## 說明

轉場效果從 A 狀態到 B 狀態，需要多少時間，就可以透過 `transition-duration` 來設定，預設值是 `0s`(0秒)。

## 範例

全部一併設定：

```css
transition-property: border-radius, width, height;
transition-duration: 2s;
```

各別設定：

```css
transition-property: border-radius, width, height;
transition-duration: 5s, 1s, 1s; /* 各別設定 */
```
